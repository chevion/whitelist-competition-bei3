## 1. 产品概述

"安全小象"是一款面向中小学生及家庭的安全教育互动平台，以IP角色"安全小象"贯穿全流程，整合逃生演练、安全问答、物资规划、我家计划四大模块，通过生成式AI动态生成个性化内容，让安全教育从"被动接受"变为"主动参与"。

- 目标用户：中小学生（6-15岁）及其家庭，校园/社区安全教育场景
- 核心价值：基于用户省份和灾害风险生成个性化安全训练内容，AI驱动动态关卡与复盘，实现"千人千面"的安全教育体验。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学生/家庭用户 | 无需注册，直接使用（本地存储配置） | 选择省份、配置AI密钥、使用全部四大模块功能 |
| 访客 | 无需注册 | 浏览基础内容 |

### 2.2 功能模块

1. **首页/省份选择页**：IP形象展示、省份快速切换、四大模块入口导航。
2. **模块一：逃生演练**：
    - **场景选择**：提供学校教室、医院、电影院等典型场景。
    - **AI动态关卡**：基于省份和场景，由AI生成障碍物和道具分布，动态修改默认地图。
    - **2D逃生游戏**：Canvas 2D引擎，支持四方向移动（WASD/方向键/移动端虚拟键），包含火灾蔓延特效、生命值系统、倒计时、物资收集（钥匙、防护服、创可贴等）。
    - **AI复盘报告**：根据用户用时、犯错记录（如走进火区）、收集物品情况，生成个性化复盘分析。
3. **模块二：安全问答**：
    - **情景化出题**：AI根据用户所在省份的常见灾害知识点生成情景题目。
    - **答题互动**：实时反馈（对/错），安全小象表情同步变化。
    - **AI解析扩展**：答题后AI生成深度解析和"小象补充知识"。
4. **模块三：物资规划**：
    - **家庭信息收集**：人数（老人/儿童/婴儿）、健康状况（慢性病）、住房类型、防范灾害。
    - **智能计算**：根据《家庭应急物资储备建议清单》标准自动计算所需物资数量。
    - **结果展示**：分类展示物资清单（饮水、食品、医疗、工具等），提供AI生成的温馨提示。
    - **导出分享**：支持将物资清单导出为长图片。
5. **模块四：我家计划**：
    - **家庭逃生图**：交互式Canvas绘图工具（绘制墙、门、窗、家具等），标记逃生出口，一键计算A*逃生路径，AI生成路线行动指南，支持导出。
    - **医疗应急卡**：填写个人病史与紧急联系人（数据纯本地处理），AI生成核心警示语和给救援者的话，支持生成卡片图片并打印。

## 3. 核心流程

### 3.1 主流程

用户访问 → (可选)设置API Key/省份 → 选择功能模块 → 完成交互任务 → 查看AI生成结果 → 导出图片/打印。

### 3.2 逃生演练流程

选择场景 → AI请求生成关卡JSON → 地图初始化（合并AI数据与模板） → 开始游戏（避障、灭火/拾取、寻找出口） → 游戏结束 → AI请求生成复盘报告。

### 3.3 安全问答流程

加载初始题库 → 用户答题 → AI请求解析与扩展 → (每3题)AI请求生成新题目 → 累计成绩 → 最终评价。

### 3.4 物资规划流程

填写表单 → 逻辑引擎计算(utils) → 展示分类清单 → 导出图片。

### 3.5 我家计划流程

- **逃生图**：绘图 → 标记出口 → A*计算最优路径 → AI生成指南文本 → 导出含指南的完整图。
- **医疗卡**：填写隐私信息 → AI生成警示/救援文案 → 生成预览图 → 打印过塑。

## 4. 用户界面设计

### 4.1 设计风格

- **色彩**：品牌橙（#FF6B35）、安全绿（#2ECC71）、警示红（#EF4444）、背景米白（#FDF8F3）。
- **组件**：Framer Motion 驱动的卡片悬停缩放、淡入淡出动效；圆角半径 1.5rem。
- **IP集成**：安全小象在不同状态（思考、开心、担心、难过）下有不同表情。

### 4.2 响应式适配

- **PC端**：侧边栏/左右分栏布局。
- **移动端**：适配底部导航，游戏界面开启虚拟摇杆，表单堆叠。

## 5. 隐私与安全

- **数据脱敏**：所有个人医疗信息仅在浏览器内存和本地 `localStorage` 处理，绝不上传服务器。
- **AI安全**：使用提示词工程确保AI回复内容符合安全教育规范，对AI生成内容标注"仅供参考"。
- **API保护**：用户自备 API Key，直连模型供应商，不经过第三方代理（增强私密性）。
